# smart-label-tool 🏷️

An intelligent data annotation tool designed to streamline the labeling process for AI and Machine Learning datasets.

## 🚀 Overview
This tool provides a user-friendly interface for annotating images and video. It aims to reduce manual effort by incorporating smart features for computer vision training.

## ✨ Features
* **Bounding Boxes & Polygons:** Tools for object detection datasets.
* **Export Formats:** Support for common formats like YOLO and COCO.
* **Workflow Optimization:** Designed for rapid data labeling and review.

## 🛠️ Tech Stack
* **Language:** Python
* **Platform:** Web-based interface
